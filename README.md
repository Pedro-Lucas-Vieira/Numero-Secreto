Jogo do Número Secreto: Aplicação web simples que gera um número aleatório entre 1 e 10 e desafia o usuário a adivinhá-lo. Desenvolvido com HTML, CSS e JavaScript.
Funcionalidades:

Geração de número aleatório sem repetição até esgotar o intervalo (1–10)
Contador de tentativas com feedback dinâmico ("maior"/"menor")
Botão de reiniciar habilitado apenas após acertar o número
Layout responsivo
